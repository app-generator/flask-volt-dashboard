How to contribute

  • 1: Select the source language: Ndyuka (Aukaans), English or French
  • 2: Select the target language, different from the source
  • 3: Provide the text to translate
  • 4: Click on Translate translate the text.
  • 5: If the translation is acceptable,
    • a: Click on I'm happy with the translation
    • b. Review the information on the pop-up screen
    • c. If you choose to leave a comment, toggle on the leave a comment button and type in your comment. The comment will not be saved if you toggle off the button
    • d. Click Save to submit the translation
  • 5: If the translation is not acceptable,
    • a: Click on I'm unhappy with the translation
    • b. Review the translation on the pop-up screen
    • c. If you choose to leave a comment, toggle on the leave a comment button and type in your comment. The comment will not be saved if you toggle off the button
    • d. Click Review to submit the translation
